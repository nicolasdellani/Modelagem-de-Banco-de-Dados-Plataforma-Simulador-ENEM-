# Modelagem-de-Banco-de-Dados-Plataforma-Simulador-ENEM-
🎓 Plataforma educacional estilo B2B2C para simulados do ENEM, desenvolvida em PostgreSQL. O projeto inclui modelagem relacional, UUIDs, JSONB, Row Level Security (RLS), triggers automáticas, controle de sessões, respostas e gerenciamento de escolas, alunos e administradores.
