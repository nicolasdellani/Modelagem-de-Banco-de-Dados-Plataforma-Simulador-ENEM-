# 🎓 Simulador ENEM — Modelagem de Banco de Dados

Projeto acadêmico desenvolvido para a disciplina de Banco de Dados, com foco na modelagem relacional de uma plataforma educacional no estilo B2B2C para aplicação de simulados do ENEM.

O sistema foi projetado utilizando PostgreSQL e contempla conceitos avançados de modelagem, segurança e integridade de dados, incluindo:

* Modelagem Entidade-Relacionamento (DER)
* Estrutura multitenancy para escolas parceiras
* Controle de acesso com Row Level Security (RLS)
* Uso de UUIDs como identificadores principais
* Relacionamentos complexos e regras de negócio
* Triggers automáticas para inicialização de perfis
* Armazenamento flexível com JSON/JSONB
* Controle de sessões de simulados e respostas
* Restrições de integridade e automações no banco

O projeto simula uma plataforma onde escolas podem cadastrar alunos, aplicar simulados e acompanhar o desempenho dos estudantes de forma segura e escalável.

## 🛠 Tecnologias Utilizadas

* PostgreSQL
* SQL
* JSONB
* UUID
* Row Level Security (RLS)

## 📚 Objetivos do Projeto

Este trabalho teve como objetivo aplicar conceitos reais de engenharia de software e banco de dados, desenvolvendo uma estrutura robusta, segura e preparada para crescimento em ambientes compartilhados.
